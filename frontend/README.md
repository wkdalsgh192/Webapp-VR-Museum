# Frontend

## Project setup

> 자바스크립트 매니저로 yarn을 사용

```
# npm을 통해 yarn을 설치
npm install --global yarn
```

- package.json에 작성된 모듈을 다운로드한다

```
yarn install
```

### Compiles and hot-reloads for development

- 로컬에서 실행시 3000 port로 접근 가능하다

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
